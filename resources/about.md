[(run alejandro)](https://alejandro.run) is a blog about software development by [Alejandro Gómez](https://github.com/purrgrammer). Alejandro is a software engineer with 10 years of experience building web servers and clients in several languages using functional programming. The topics treated here include functional programming, web development, Clojure(Script) and databases.
